# Claw Personal Site v2

## 目标
将产品风的企业主页转变为个人风格的主页，参考 pixelwrld.co 的设计风格。

## 关键变更
1. 移除 3D Lobster 和粒子背景
2. 使用简洁的个人头像区域
3. 简化为个人主页风格
4. 保留 1.0 版本入口
5. 更简洁的布局和动画

## 设计参考
- pixelwrld.co - 米白背景、简洁排版、优雅动画
- 配色：米白 #FAF9F6、黑色 #1A1A1A、橙色点缀 #FF6B35

## 页面结构
- Hero：头像 + 简短自我介绍
- 关于：个人故事/随笔
- 项目：精选项目展示
- 底部：社交链接 + 1.0 版本入口

## 技术栈
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion (简化动画)
